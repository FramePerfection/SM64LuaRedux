# SM64 Lua Redux

SM64 Lua Redux is a fork of [SM64 Lua](https://github.com/mkdasher/SM64Lua) powered by [mupen-lua-ugui](https://github.com/Aurumaker72/mupen-lua-ugui)

# Features

- Compatible with >=1.1.4
- Customizable visual styles
- Customizable layout
